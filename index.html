The Luhn Algorithm

To find out if a credit card number is valid or not, we can use the Luhn algorithm.

Generally speaking, an algorithm is a series of steps that solve a problem.

The Luhn algorithm is a series of mathematical calculations used to validate certain identification numbers, e.g. credit card numbers.
